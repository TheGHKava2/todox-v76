# ToDoX Web (v5)
Front-end em Next.js 14 (App Router) + Tailwind, com módulos:
- Dashboard, Backlog (tabela), Board (colunas), YAML Studio (Monaco), Scaffold Hub e Watcher UI, PRs.

## Rodar
```bash
npm install
npm run dev
# http://localhost:3000 (usa NEXT_PUBLIC_API_URL ou http://localhost:8000 por padrão)
```
Defina `NEXT_PUBLIC_API_URL` se a API não estiver em `http://localhost:8000`.
